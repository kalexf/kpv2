Welcome to Kanplan

This web app let's you build customised trianing plans for running and other forms exercise

Create new activity types

The first step is to create activities, from which you can then build a plan. Select a run type and then enter the details. For most training types you have the option of choosing to mkae the activity progressive or static.

Building a plan

Once you have cre3ated some activity types you can build a plan by selecting which activities to perform on which says. The default plan length is 2 weeks per cycle, this can be changed in preferences.

The home screen

Once you ahve created and saved a plan, you will see your schedule of acitvities displayed on the home screen. here you can see (etc..) explain submit

Updates
When you log in you will be shown an update screen that asks you to confirm whether you ahve completed exercises that were scheduled between now and your last login. If you did the acitivities select completed, if not select did not complete.

Activity / distance hsitory

These pages provide a history of your recently performed activities, and your total weekly distance per week.

Preferences

On the preferences there are various options for adjusting your plan length, deleting activity history, or changing your pace settings
